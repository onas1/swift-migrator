## 3. Installation

### Downloading Releases
Visit **GitHub Releases** and download the appropriate ZIP for your operating system.

Example structure:
```
migrator-win-x64.zip
migrator-linux-x64.tar.gz
migrator-macos-arm64.tar.gz
```

Extract the archive and place `migrator` somewhere in your PATH.

### Example
#### Windows
```
C:\tools\migrator\migrator.exe
```

#### Linux / macOS
```
sudo mv migrator /usr/local/bin/migrator
chmod +x /usr/local/bin/migrator
```
